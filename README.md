## How to setup EmailJS (Required for email feature)

1. Go to emailjs.com and create free account
2. Click "Add New Service" - choose Gmail
3. Go to "Email Templates" - create new template
4. Go to Account → API Keys → copy Public Key
5. Open script.js and replace:
   - `YOUR_PUBLIC_KEY_HERE` with your actual key
   - `YOUR_SERVICE_ID` with your service ID
   - `YOUR_TEMPLATE_ID` with your template ID

## How to run this project

1. Download all 3 files (index.html, style.css, script.js)
2. Put them in the same folder
3. Double-click index.html to open in browser
4. Or use VS Code Live Server

## How to test the website

# LaundryHub - Task 20

## What I made
A laundry service website where you can book laundry services.

## What's in it
- Cart on left, services on right
- Navigation bar at top
- Skip button to hide services
- Newsletter signup
- Footer with links

## How to use
1. Add services to cart
2. Enter quantity
3. Fill the form
4. Confirm booking

## How to run
1. Download all 3 files
2. Put them in same folder
3. Open index.html in browser

## Files
- index.html - Main page
- style.css - Colors and layout
- script.js - Buttons and cart
